# Installing the First Time

- clone repo
- git pull origin main
- cd cinesync
- npm install
   
# To Start

- make sure you're in the cinesync directory (e.g. /workspaces/cinesync/cinesync)
- npx expo start
- open Expo Go app and scan QR code in terminal
